### [Project 1: Customer Trends Data Analysis (SQL, Python & Power BI)](https://github.com/Neha-Ydv/customer-trends-data-analysis-SQL-Python-PowerBI.git)
This project delivers a full end-to-end analytics workflow to understand customer behavior and purchasing trends. Using rich transaction data, I leveraged SQL to extract and segment customer cohorts, then applied Python (Pandas, NumPy, visualizations) to explore insights such as spending habits, loyalty patterns, and product affinity. Finally, I built an interactive Power BI dashboard that presents key insights to stakeholders, highlighting high-value customer segments, monthly revenue drivers, and opportunities for targeted retention strategies.

**Tools Used:** SQL, Python (Pandas, NumPy, Matplotlib/Seaborn), Power BI  

**Project Link:** [GitHub Repo](https://github.com/Neha-Ydv/customer-trends-data-analysis-SQL-Python-PowerBI.git)
